New technologies used during education in pandemic

  1. Zoom

    2. Zoom is a video conferencing program that has been used by the uni to deliver online classes.It has the ability to share video and voice to other people online, making it ideal for creating a virtual classroom environment.Furthermore there is also an option to send messages in the chat, for example when asking questions to a lecturer during a zoom session without having to unmute mic.


  2. Scanning apps on phone

    3. These apps have been useful during the submission of online written exams as they allow a student to take a picture of their answers using their phones.The written answers are converted to a collection of pdf document which can then be submitted to blackboard.


  3. Google docs

    4. This program allows for collaboration on a document with multiple users which makes it ideal for group work.Setting up a google document for group is simple as it requires creating a document and then sharing the link for other members of the group. Being able to work on a document with multiple users at a time is useful as it makes it quicker and more efficient to be able to finish any group projects.
